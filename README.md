# mr1upmachine's Stream Tools

A collection of tools I've built on top of [Streamer.bot](https://streamer.bot) and [OBS](https://obsproject.com/) for my Twitch streams at [twitch.tv/mr1upmachine](https://www.twitch.tv/mr1upmachine).

---

## Tools

### [Quote System](quote/README.md)

Twitch chat commands for managing and retrieving stream quotes, backed by Streamer.bot.

### [Chat Credits](chat-credits/README.md)

Logs chatters during a stream to generate an end-of-stream credits screen.
